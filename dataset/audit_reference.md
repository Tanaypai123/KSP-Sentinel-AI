# Audit Mode Explainability Reference

This document describes the regulatory compliance and audit trails provided by **Audit Mode** in the Explainability Engine.

## 🎯 Purpose
Support legal defense teams, administrative inspectors, and audit boards with forensic proof of how the AI made its determinations.

## 📋 Audit Mode Trace Content
- **Execution ID / Conversation ID:** Reconstruct context session bounds.
- **Reason Trace:** The complete logical step-chain generated by the reasoning engine.
- **Safety Decisions:** Full trace of hallucination checks (violations detected, categories validated, disclaimers applied).
- **Module Trace:** Raw status (executed/skipped) and latency of each pipeline stage.
